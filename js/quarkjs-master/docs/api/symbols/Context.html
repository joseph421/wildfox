Class Context


Context是Quark框架中显示对象结构的上下文，实现显示对象结构的渲染。此类为抽象类。

构造函数概览

说明
 
Context(props)
构造函数.

方法概览

说明
 
draw()
绘制显示对象，需要子类来实现。
 
endDraw()
完成绘制显示对象后的处理方法，需要子类来实现。
 
remove(target)
从画布中删除显示对象，需要子类来实现。
 
startDraw()
为开始绘制显示对象做准备，需要子类来实现。
 
transform()
对显示对象进行变换，需要子类来实现。

构造函数详细信息

Context(props)
构造函数.
参数:
{Object} props
一个对象。包含以下属性：

canvas - 渲染上下文所对应的画布。

方法详细信息

draw()
绘制显示对象，需要子类来实现。
endDraw()
完成绘制显示对象后的处理方法，需要子类来实现。
remove(target)
从画布中删除显示对象，需要子类来实现。
参数:
{DisplayObject} target
要删除的显示对象。
startDraw()
为开始绘制显示对象做准备，需要子类来实现。
transform()
对显示对象进行变换，需要子类来实现。
