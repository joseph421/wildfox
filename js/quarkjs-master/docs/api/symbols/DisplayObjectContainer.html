Class DisplayObjectContainer


继承自 DisplayObject. DisplayObjectContainer类继承自DisplayObject，是显示列表中显示对象容器的基类。每个DisplayObjectContainer对象都有自己的子级列表children，用于组织对象的Z轴顺序。注意：DisplayObjectContainer对象的宽高默认为0，在autoSize=false的情况下，需要手动设置宽高。

构造函数概览

说明
 
DisplayObjectContainer()
构造函数.

属性概览

说明
 
autoSize
指示DisplayObjectContainer是否随子元素自动设置大小。默认为false。
 
eventChildren
指示DisplayObjectContainer的子元素是否接受交互事件，如mousedown，touchstart等。默认为true。
从类 DisplayObject 继承的属性:
alpha, eventEnabled, height, id, mask, name, parent, polyArea, regX, regY, rotation, scaleX, scaleY, transformEnabled, useHandCursor, visible, width, x, y

getObjectUnderPoint(x, y, usePolyCollision, returnAll)
返回x和y指定点下的DisplayObjectContainer的子项（或孙子项，依此类推）的数组集合。默认只返回最先加入的子显示对象。
参数:
{Number} x
指定点的x轴坐标。
{Number} y
指定点的y轴坐标。
{Boolean} usePolyCollision
指定是否采用多边形碰撞检测。默认为false。
{Boolean} returnAll
指定是否返回指定点下的所有显示对象。默认为false。
返回值:
返回指定点下的显示对象集合，当然returnAll为false时只返回最先加入的子显示对象。
