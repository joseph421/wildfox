Class ImageLoader


继承自 EventDispatcher. ImageLoader类是一个图片加载器，用于动态加载图片资源。

构造函数概览

说明
 
ImageLoader(source)
构造函数.

方法概览

说明
 
getLoaded()
返回已加载图片资源的数目。
 
getLoadedSize()
返回已加载的图片资源的大小之和（在图片资源的大小size已指定的情况下）。
 
getTotal()
返回所有图片资源的总数。
 
getTotalSize()
返回所有图片资源的大小之和（在图片资源的大小size已指定的情况下）。
 
load(source)
开始顺序加载图片资源。
从类 EventDispatcher 继承的方法:
addEventListener, dispatchEvent, hasEventListener, removeAllEventListeners, removeEventListener, removeEventListenerByType

构造函数详细信息

ImageLoader(source)
构造函数.
参数:
source
要加载的图片资源，可以是一个单独资源或多个资源的数组。图片资源格式为：{src:$url, id:$id, size:$size}。

方法详细信息

getLoaded()
返回已加载图片资源的数目。
getLoadedSize()
返回已加载的图片资源的大小之和（在图片资源的大小size已指定的情况下）。
getTotal()
返回所有图片资源的总数。
getTotalSize()
返回所有图片资源的大小之和（在图片资源的大小size已指定的情况下）。
load(source)
开始顺序加载图片资源。
参数:
source
要加载的图片资源，可以是一个单独资源或多个资源的数组。
