Class Quark


Quark是QuarkJS框架的全局对象，也是框架内部所有类的全局命名空间。在全局Q未被占用的情况下，也可以使用其缩写Q。

<static> {Object} Quark.cacheObject(obj, toImage, type)
把DisplayObject对象绘制到一个新的画布上。可作为缓存使用，也可转换成dataURL格式的位图。
Defined in: Utils.js.
参数:
{DisplayObject} obj
要缓存的显示对象。
{Boolean} toImage
指定是否把缓存转为DataURL格式的。默认为false。
{String} type
指定转换为DataURL格式的图片mime类型。默认为"image/png"。
返回值:
{Object} 显示对象的缓存结果。根据参数toImage不同而返回Canvas或Image对象。

{Number} y

指定是否采用多边形碰撞。默认为false。
返回值:
{Number} 如果点x，y在对象obj内为1，在外为-1，在边上为0。
