Class EventManager


EventManager是一个简单的系统事件管理器。

构造函数概览

说明
 
EventManager()
构造函数.

方法概览

说明
 
register(target, events, callback, preventDefault, stopPropagation)
注册DOM事件侦听，当事件触发时调用callback函数。
 
registerStage(stage, events, preventDefault, stopPropagation)
注册Quark.Stage事件侦听，使得Stage能够接收和处理指定的事件。
<static>  
EventManager.stop(e, continueDefault, continuePropagation)
停止事件。
 
unregister(target, events, callback)
删除对象事件侦听。
 
unregisterStage(stage, events)
删除Quark.Stage事件侦听。

构造函数详细信息

EventManager()
构造函数.

方法详细信息

register(target, events, callback, preventDefault, stopPropagation)
注册DOM事件侦听，当事件触发时调用callback函数。
参数:
target
事件目标DOM对象。
events
要注册事件类型数组。
callback
preventDefault
stopPropagation
registerStage(stage, events, preventDefault, stopPropagation)
注册Quark.Stage事件侦听，使得Stage能够接收和处理指定的事件。
参数:
stage
Quark.Stage舞台对象。
events
要注册的事件类型数组。
preventDefault
stopPropagation
<static> EventManager.stop(e, continueDefault, continuePropagation)
停止事件。
参数:
e
要停止的事件对象。
continueDefault
是否继续事件的默认行为。
continuePropagation
是否继续事件的冒泡。
unregister(target, events, callback)
删除对象事件侦听。
参数:
target
事件目标DOM对象。
events
要删除的事件类型数组。
callback
unregisterStage(stage, events)
删除Quark.Stage事件侦听。
参数:
stage
Quark.Stage舞台对象。
events
要删除的事件类型数组。
