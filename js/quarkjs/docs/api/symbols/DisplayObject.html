Class DisplayObject


继承自 EventDispatcher. DisplayObject类是可放在舞台上的所有显示对象的基类。DisplayObject类定义了若干显示对象的基本属性。渲染一个DisplayObject其实是进行若干变换后再渲染其drawable对象。

构造函数概览

说明
 
DisplayObject()
构造函数.

属性概览

说明
 
alpha
DisplayObject对象的透明度。取值范围为0-1，默认为1。
 
eventEnabled
指示DisplayObject对象是否接受交互事件，如mousedown，touchstart等。默认为true。
 
height
DisplayObject对象的高。
 
id
DisplayObject对象唯一标识符id。
 
mask
指示DisplayObject对象的遮罩对象。当上下文为DOMContext时暂时只支持webkit内核浏览器。默认为null。
 
name
DisplayObject对象的名称。
 
parent
DisplayObject对象的父容器。只读属性。
 
polyArea
指示DisplayObject对象的多边形碰撞区域。默认为null，即使用对象的外包围矩形。
 
regX
DisplayObject对象的注册点（中心点）的x轴坐标。
 
regY
DisplayObject对象的注册点（中心点）的y轴坐标。
 
rotation
DisplayObject对象的旋转角度。默认为0。
 
scaleX
DisplayObject对象在x轴上的缩放值。取值范围为0-1。
 
scaleY
DisplayObject对象在y轴上的缩放值。取值范围为0-1。
 
transformEnabled
指示DisplayObject对象是否执行变换。默认为false。
 
useHandCursor
指示DisplayObject对象是否支持手型的鼠标光标。默认为false。
 
visible
指示DisplayObject对象是否可见。默认为true。
 
width
DisplayObject对象的宽。
 
x
DisplayObject对象相对父容器的x轴坐标。
 
y
DisplayObject对象相对父容器的y轴坐标。

方法概览

说明
 
getBounds()
返回DisplayObject对象在舞台全局坐标系内的矩形区域以及所有顶点。
 
getCurrentHeight()
获得DisplayObject对象变形后的高度。
 
getCurrentWidth()
获得DisplayObject对象变形后的宽度。
 
getDrawable(context)
获得可绘制对象实体，如Image或Canvas等其他DOM对象。
 
getStage()
获得DisplayObject对象的舞台引用。如未被添加到舞台，则返回null。
 
getState(propName)
获得DisplayObject对象保存的状态列表中的指定的属性状态。
 
globalToLocal(x, y)
将x和y指定的点从舞台（全局）坐标转换为显示对象的（本地）坐标。
 
hitTestObject(object, usePolyCollision)
计算DisplayObject对象的包围矩形，以确定由object参数指定的显示对象是否与其相交。
 
hitTestPoint(x, y, usePolyCollision)
计算DisplayObject对象的包围矩形，以确定由x和y参数指定的点是否在其包围矩形之内。
 
localToGlobal(x, y)
将x和y指定的点从显示对象的（本地）坐标转换为舞台（全局）坐标。
 
localToTarget(x, y, target)
将x和y指定的点从显示对象的（本地）坐标转换为指定对象的坐标系里坐标。
 
propChanged(prop)
比较DisplayObject对象的当前状态和最近一次保存的状态，返回指定属性中是否发生改变。
 
render(context)
DisplayObject对象渲染接口，可通过覆盖此方法实现对象的渲染。
 
saveState(list)
保存DisplayObject对象的状态列表中的各种属性状态。
 
setDrawable(drawable)
设置可绘制对象，默认是一个Image对象，可通过覆盖此方法进行DOM绘制。
 
toString()
返回DisplayObject对象的全路径的字符串表示形式，方便debug。如Stage1.Container2.Bitmap3。
 
update(timeInfo)
对象数据更新接口，可通过覆盖此方法实现对象的数据更新。
从类 EventDispatcher 继承的方法:
addEventListener, dispatchEvent, hasEventListener, removeAllEventListeners, removeEventListener, removeEventListenerByType

方法详细信息

{Object} getBounds()
返回DisplayObject对象在舞台全局坐标系内的矩形区域以及所有顶点。
返回值:
{Object} 返回显示对象的矩形区域。
{Number} getCurrentHeight()
获得DisplayObject对象变形后的高度。
返回值:
{Number} 返回对象变形后的高度。
{Number} getCurrentWidth()
获得DisplayObject对象变形后的宽度。
返回值:
{Number} 返回对象变形后的宽度。
getDrawable(context)
获得可绘制对象实体，如Image或Canvas等其他DOM对象。
参数:
{Context} context
渲染上下文。
{Stage} getStage()
获得DisplayObject对象的舞台引用。如未被添加到舞台，则返回null。
返回值:
{Stage} 返回对象的舞台。
getState(propName)
获得DisplayObject对象保存的状态列表中的指定的属性状态。
参数:
{String} propName
要获取的属性状态名称。
返回值:
返回指定属性的最后一次保存状态值。
{Object} globalToLocal(x, y)
将x和y指定的点从舞台（全局）坐标转换为显示对象的（本地）坐标。
参数:
{Number} x
显示对象的全局x轴坐标。
{Number} y
显示对象的全局y轴坐标。
返回值:
{Object} 返回转换后的本地坐标对象。格式如：{x:10, y:10}。
{Boolean} hitTestObject(object, usePolyCollision)
计算DisplayObject对象的包围矩形，以确定由object参数指定的显示对象是否与其相交。
参数:
{DisplayObject} object
指定检测碰撞的显示对象。
{Boolean} usePolyCollision
指定是否采用多边形碰撞。默认为false。
返回值:
{Boolean} 相交返回true，否则返回false。
{Number} hitTestPoint(x, y, usePolyCollision)
计算DisplayObject对象的包围矩形，以确定由x和y参数指定的点是否在其包围矩形之内。
参数:
{Number} x
指定碰撞点的x坐标。
{Number} y
指定碰撞点的y坐标。
{Boolean} usePolyCollision
指定是否采用多边形碰撞。默认为false。
返回值:
{Number} 在包围矩形之内返回1，在边界上返回0，否则返回-1。
{Object} localToGlobal(x, y)
将x和y指定的点从显示对象的（本地）坐标转换为舞台（全局）坐标。
参数:
{Number} x
显示对象的本地x轴坐标。
{Number} y
显示对象的本地y轴坐标。
返回值:
{Object} 返回转换后的全局坐标对象。格式如：{x:10, y:10}。
{Object} localToTarget(x, y, target)
将x和y指定的点从显示对象的（本地）坐标转换为指定对象的坐标系里坐标。
参数:
{Number} x
显示对象的本地x轴坐标。
{Number} y
显示对象的本地y轴坐标。
target
返回值:
{Object} 返回转换后指定对象的本地坐标对象。格式如：{x:10, y:10}。
propChanged(prop)
比较DisplayObject对象的当前状态和最近一次保存的状态，返回指定属性中是否发生改变。
参数:
prop
可以是单个或多个属性参数。
返回值:
属性改变返回true，否则返回false。
render(context)
DisplayObject对象渲染接口，可通过覆盖此方法实现对象的渲染。
参数:
{Context} context
渲染上下文。
saveState(list)
保存DisplayObject对象的状态列表中的各种属性状态。
参数:
{Array} list
要保存的属性名称列表。默认为null。
setDrawable(drawable)
设置可绘制对象，默认是一个Image对象，可通过覆盖此方法进行DOM绘制。
参数:
{Object} drawable
要设置的可绘制对象。一般是一个Image对象。
{String} toString()
返回DisplayObject对象的全路径的字符串表示形式，方便debug。如Stage1.Container2.Bitmap3。
返回值:
{String} 返回对象的全路径的字符串表示形式。如Stage1.Container2.Bitmap3。
{Boolean} update(timeInfo)
对象数据更新接口，可通过覆盖此方法实现对象的数据更新。
参数:
{Object} timeInfo
对象更新所需的时间信息。
返回值:
{Boolean} 更新成功返回true，否则为false。
