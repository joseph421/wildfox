Class Button


继承自 DisplayObjectContainer. Button类继承自DisplayObjectContainer，是Quark中的简单按钮实现。

构造函数详细信息

Button()
构造函数.
参数:
{Object}
props 一个对象，包含以下属性：

image - Image对象。

up - 按钮弹起状态下的显示帧数组对象。如：[0,0,50,50]。

over - 按钮经过状态下的显示帧数组对象。如：[50,0,50,50]。

down - 按钮按下状态下的显示帧数组对象。如：[100,0,50,50]。

disabled - 按钮不可用状态下的显示帧数组对象。如：[150,0,50,50]。
