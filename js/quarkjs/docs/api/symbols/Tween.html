Class Tween


Tween类是一个缓动动画类。使用它能实现移动、改变大小、淡入淡出等效果。

构造函数概览

说明
 
Tween(target, newProps, params)
构造函数.

方法概览

说明
<static>  
Tween.add(tween)
添加Tween实例。
<static>  
Tween.from(target, toProps, params)
创建一个缓动动画，让目标对象从指定的起始属性变换到当前属性。
 
pause()
暂停缓动动画的播放。
<static>  
Tween.remove(tween)
删除Tween实例。
 
resume()
恢复缓动动画的播放。
 
setProps(oldProps, newProps)
设置缓动对象的初始和目标属性。
 
start()
启动缓动动画的播放。
<static>  
Tween.step()
更新所有Tween实例，一般由Quark.Timer类自动调用。
 
stop()
停止缓动动画的播放。
<static>  
Tween.to(target, toProps, params)
创建一个缓动动画，让目标对象从当前属性变换到目标属性。
