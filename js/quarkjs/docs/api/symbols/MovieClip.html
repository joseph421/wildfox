Class MovieClip


继承自 Bitmap. MovieClip影片剪辑类，表示一组动画片段。MovieClip是由Image对象的若干矩形区域组成的集合序列，并按照一定规则顺序播放。帧frame的定义格式为：{rect:*required*, label:"", interval:0, stop:0, jump:-1}。

说明
 
MovieClip()
构造函数.
从类 DisplayObject 继承的属性:
alpha, eventEnabled, height, id, mask, name, parent, polyArea, regX, regY, rotation, scaleX, scaleY, transformEnabled, useHandCursor, visible, width, x, y

方法概览

说明
 
addFrame(frame)
向MovieClip中添加帧frame，可以是单个帧或多帧的数组。
 
getFrame(indexOrLabel)
获得指定位置或标签的帧frame。
 
getFrameIndex(indexOrLabel)
获得指定参数对应的帧的位置。
 
getNumFrames()
返回MovieClip的帧数。
 
gotoAndPlay(indexOrLabel)
跳转到指定位置或标签的帧，并继续播放动画序列。
 
gotoAndStop(indexOrLabel)
跳转到指定位置或标签的帧，并停止播放动画序列。
 
nextFrame(displayedDelta)
播放动画序列的下一帧。
 
play()
从当前位置开始播放动画序列。
 
render(context)
渲染当前帧到舞台。
 
setFrame(frame, index)
指定帧frame在MovieClip的播放序列中的位置（从0开始）。
 
stop()
停止播放动画序列。
