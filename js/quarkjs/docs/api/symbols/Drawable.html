Class Drawable


Drawable是可绘制图像或DOM的包装。当封装的是HTMLImageElement、HTMLCanvasElement或HTMLVideoElement对象时，可同时支持canvas和dom两种渲染方式，而如果封装的是dom时，则不支持canvas方式。

构造函数概览

说明
 
Drawable(drawable, isDOM)
构造函数.

方法概览

说明
 
get(obj, context)
根据context上下文获取不同的Drawable包装的对象。
 
set(drawable, isDOM)
设置Drawable对象。

构造函数详细信息

Drawable(drawable, isDOM)
构造函数.
参数:
drawable
一个可绘制对象。
{Boolean} isDOM
指定参数drawable是否为一个DOM对象。默认为false。

方法详细信息

get(obj, context)
根据context上下文获取不同的Drawable包装的对象。
参数:
{DisplayObject} obj
指定的显示对象。
{Context} context
指定的渲染上下文。
返回值:
返回包装的可绘制对象。
set(drawable, isDOM)
设置Drawable对象。
参数:
drawable
一个可绘制对象。
{Boolean} isDOM
指定参数drawable是否为一个DOM对象。默认为false。
