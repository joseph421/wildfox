Class CanvasContext


继承自 Context. CanvasContext是Canvas渲染上下文，将显示对象渲染到指定的Canvas上。

构造函数概览

说明
 
CanvasContext(props)
构造函数.

方法概览

说明
 
clear(x, y, width, height)
清除画布上的指定区域内容。
 
draw(target)
绘制指定的显示对象到Canvas上。
 
endDraw()
绘制完毕，恢复上下文。
 
startDraw()
准备绘制，保存当前上下文。
 
transform(target)
对指定的显示对象进行context属性设置或变换。
从类 Context 继承的方法:
remove

构造函数详细信息

CanvasContext(props)
构造函数.
参数:
{Object} props
一个对象。包含以下属性：

canvas - 渲染上下文所对应的canvas，HTMLCanvasElement对象。

方法详细信息

clear(x, y, width, height)
清除画布上的指定区域内容。
参数:
{Number} x
指定区域的x轴坐标。
{Number} y
指定区域的y轴坐标。
{Number} width
指定区域的宽度。
{Number} height
指定区域的高度。
draw(target)
绘制指定的显示对象到Canvas上。
参数:
{DisplayObject} target
要绘制的显示对象。
endDraw()
绘制完毕，恢复上下文。
startDraw()
准备绘制，保存当前上下文。
transform(target)
对指定的显示对象进行context属性设置或变换。
参数:
{DisplayObject} target
要进行属性设置或变换的显示对象。
