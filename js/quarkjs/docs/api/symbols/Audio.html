Class Audio


Audio类是原生Audio的封装。

构造函数概览

说明
 
Audio(src, preload, autoPlay, loop)
构造函数.

方法概览

说明
 
load()
开始加载声音文件。
 
loaded()
指示声音文件是否已被加载。
 
play()
开始播放。
 
playing()
指示声音是正在播放。
 
stop()
停止播放。

构造函数详细信息

Audio(src, preload, autoPlay, loop)
构造函数.
参数:
src
要加载的声音的地址。
preload
指示是否自动加载，在某些浏览器下无效，如IOS上的Safari。
autoPlay
指示是否自动播放，在某些浏览器下无效，如IOS上的Safari。
loop
指示是否循环播放。

方法详细信息

load()
开始加载声音文件。
loaded()
指示声音文件是否已被加载。
play()
开始播放。
playing()
指示声音是正在播放。
stop()
停止播放。
