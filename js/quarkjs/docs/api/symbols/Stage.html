Class Stage


继承自 DisplayObjectContainer. 舞台是显示对象的根，所有显示对象都会被添加到舞台上，必须传入一个context使得舞台能被渲染。舞台是一种特殊显示对象容器，可以容纳子显示对象。

构造函数概览

说明
 
Stage()
构造函数.

属性概览

说明
 
paused
指示舞台更新和渲染是否暂停。默认为false。
 
stageX
舞台在页面中的X偏移量，即offsetLeft。只读属性。可通过调用updatePosition()方法更新。
 
stageY
舞台在页面中的Y偏移量，即offsetTop。只读属性。可通过调用updatePosition()方法更新。
从类 DisplayObjectContainer 继承的属性:
autoSize, eventChildren
从类 DisplayObject 继承的属性:
alpha, eventEnabled, height, id, mask, name, parent, polyArea, regX, regY, rotation, scaleX, scaleY, transformEnabled, useHandCursor, visible, width, x, y

方法概览

说明
 
dispatchEvent(e)
舞台Stage默认的事件处理器。
 
step(timeInfo)
更新舞台Stage上的所有显示对象。可被Quark.Timer对象注册调用。
 
updatePosition()
更新舞台Stage在页面中的偏移位置，即stageX/stageY。
从类 DisplayObjectContainer 继承的方法:
addChild, addChildAt, contains, getChildAt, getChildById, getChildIndex, getNumChildren, getObjectUnderPoint, removeAllChildren, removeChild, removeChildAt, removeChildById, render, setChildIndex, sortChildren, swapChildren, swapChildrenAt
从类 DisplayObject 继承的方法:
getBounds, getCurrentHeight, getCurrentWidth, getDrawable, getStage, getState, globalToLocal, hitTestObject, hitTestPoint, localToGlobal, localToTarget, propChanged, saveState, setDrawable, toString, update
从类 EventDispatcher 继承的方法:
addEventListener, hasEventListener, removeAllEventListeners, removeEventListener, removeEventListenerByType
