Class Graphics


继承自 DisplayObject. Graphics类包含一组创建矢量图形的方法。

构造函数概览

说明
 
Graphics()
构造函数.
从类 DisplayObject 继承的属性:
alpha, eventEnabled, height, id, mask, name, parent, polyArea, regX, regY, rotation, scaleX, scaleY, transformEnabled, useHandCursor, visible, width, x, y

方法概览

说明
 
beginBitmapFill(image, repetition)
开始一个位图填充样式。
 
beginFill(fill, alpha)
指定绘制图形的填充样式和透明度。
 
beginLinearGradientFill(x0, y0, x1, y1, colors, ratios)
指定绘制图形的线性渐变填充样式。
 
beginPath()
开始一个新的路径。
 
beginRadialGradientFill(x0, y0, r0, x1, y1, r1, colors, ratios)
指定绘制图形的放射性渐变填充样式。
 
cache(toImage)
缓存graphics到一个canvas或image。可用来提高渲染效率。
 
clear()
清除所有绘制动作并复原所有初始状态。
 
closePath()
关闭当前的路径。
 
drawCircle(x, y, radius)
绘制一个圆。
 
drawEllipse(x, y, width, height)
绘制一个椭圆。
 
drawRect(x, y, width, height)
绘制一个矩形。
 
drawRoundRect(x, y, width, height, cornerSize)
绘制一个圆角矩形。
 
drawRoundRectComplex(x, y, width, height, cornerTL, cornerTR, cornerBR, cornerBL)
绘制一个复杂的圆角矩形。
 
drawSVGPath(pathData)
根据参数指定的SVG数据绘制一条路径。 代码示例:

var path = "M250 150 L150 350 L350 350 Z";

var shape = new Quark.Graphics({width:500, height:500});

shape.drawSVGPath(path).beginFill("#0ff").endFill();
 
endFill()
应用并结束笔画的绘制和图形样式的填充。
 
lineStyle(thickness, lineColor, alpha, lineCap, lineJoin, miterLimit)
指定绘制图形的线条样式。
 
toImage(type)
把Graphics对象转换成dataURL格式的位图。
 
uncache()
清除缓存。
从类 DisplayObject 继承的方法:
getBounds, getCurrentHeight, getCurrentWidth, getDrawable, getStage, getState, globalToLocal, hitTestObject, hitTestPoint, localToGlobal, localToTarget, propChanged, render, saveState, setDrawable, toString, update
从类 EventDispatcher 继承的方法:
addEventListener, dispatchEvent, hasEventListener, removeAllEventListeners, removeEventListener, removeEventListenerByType
