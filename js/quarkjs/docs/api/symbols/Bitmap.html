Class Bitmap


继承自 DisplayObject. Bitmap位图类，表示位图图像的显示对象，简单说它就是Image对象的某个区域的抽象表示。

构造函数概览

说明
 
Bitmap()
构造函数.
从类 DisplayObject 继承的属性:
alpha, eventEnabled, height, id, mask, name, parent, polyArea, regX, regY, rotation, scaleX, scaleY, transformEnabled, useHandCursor, visible, width, x, y

方法概览

说明
 
render(context)
覆盖父类的渲染方法。渲染image指定的显示区域。
 
setRect(rect)
设置Bitmap对象的image的显示区域。
从类 DisplayObject 继承的方法:
getBounds, getCurrentHeight, getCurrentWidth, getDrawable, getStage, getState, globalToLocal, hitTestObject, hitTestPoint, localToGlobal, localToTarget, propChanged, saveState, setDrawable, toString, update
从类 EventDispatcher 继承的方法:
addEventListener, dispatchEvent, hasEventListener, removeAllEventListeners, removeEventListener, removeEventListenerByType

构造函数详细信息

Bitmap()
构造函数.
参数:
{Object}
props 一个对象，包含以下属性：

image - Image对象。

rect - Image对象的矩形区域。格式为：[0,0,100,100]

方法详细信息

render(context)
覆盖父类的渲染方法。渲染image指定的显示区域。
参数:
{Context} context
渲染上下文。
setRect(rect)
设置Bitmap对象的image的显示区域。
参数:
{Array} rect
要设置的显示区域数组。格式为：[rectX, rectY, rectWidth, rectHeight]。
